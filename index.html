VMware Guest Console Tool

GUI Tool to launch the console of a powered on VM Guest.

Currently, only powered on guests are included, although, if requested, a version to include all guests can be compiled. This takes into account only reporting on active VMs and not VMs that may be part of SRM, Veaam B&R, etc.

Current Version - 1.1.0

Updates:

  • Added connected server name in form window
  • Now allows multiple instances of the application to be launched
  • Added message once connected that reminds the user to select a VM from the drop-down VM list
  • Minor code and variable cleanup

Minimum Requirements:

  • PowerCLI 5.5
  • .NET Framework 4.0
  • PowerShell 3.0
  • VMware Client Integration Plugin must be installed
  • Guest Console privileges to at least 1 VM managed by a reachable vCenter Server
